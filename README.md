# Проект веб-сервиса "FitHit" для генерации индивидуального плана питания 

FitHit — это цифровой помощник в области здорового питания, предназначенный для широкого круга пользователей, заинтересованных в индивидуальном подходе к организации своего рациона и улучшении образа жизни.
На основе антропометрических данных (рост, вес, возраст, пол, уровень активности) и целей (поддержание, похудение, набор массы) пользователя сервис рассчитывает суточную норму КБЖУ (калории, белки, жиры, углеводы) и составляет индивидуальный и подробный план питания.
